## A/B Testing Analysis 📊
Overview

This project performs A/B Testing Analysis on a marketing dataset to evaluate ad performance and determine which version performs better in terms of conversions.

The analysis includes data cleaning, exploratory data analysis (EDA), statistical testing, and visualization to generate actionable insights.

## Objectives 🎯
Compare Ad A vs Ad B performance
Analyze conversion rates
Perform statistical hypothesis testing
Identify factors affecting conversions
Generate data-driven insights
